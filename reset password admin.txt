chỉ cần chạy đoạn này trong Console của DevTools (F12 → Console):
const users = JSON.parse(localStorage.getItem('flashlearn_users') || '[]');
const idx = users.findIndex(u => u.username === 'admin');
if (idx !== -1) {
  users[idx].password = 'admin123';
  localStorage.setItem('flashlearn_users', JSON.stringify(users));
  console.log('✓ Đã đổi mật khẩu admin thành: admin123');
} else {
  console.log('✗ Không tìm thấy tài khoản admin');
}
Sau đó refresh trang và đăng nhập với:

Username: admin
Password: admin123
